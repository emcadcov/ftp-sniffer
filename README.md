### FTP-sniffer

Script cathes ftp packets and print time, source and destination IPs, ftp command and packet length.

### Usage
You must have jNetPcap library and must be superuser to work with wireless adapter on your device.

### Logs example
-----------------------------
Wed Dec 02 19:13:02 MSK 2015

Length: 96

Destination IP: 172.17.48.255
Source IP: 94.198.43.90

Info:
230 Anonymous user logged in
